# Gamified Environmental Education Platform

A Spring Boot web application that teaches environmental awareness through interactive games.

## Features

- Eco Quiz
- Eco Memory Game
- Waste Sorting Game
- Leaderboard
- Points and Badges

## Tech Stack

Backend
- Spring Boot
- MySQL
- REST API

Frontend
- Thymeleaf
- HTML
- CSS
- JavaScript

## Games

1. Environmental Quiz
2. Memory Match Game
3. Waste Sorting Game

## Run the Project

1. Clone the repository
2. Run Spring Boot application
3. Open

http://localhost:8080
